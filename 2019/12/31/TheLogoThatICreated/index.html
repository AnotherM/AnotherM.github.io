那些年我创造过的Logo

发表于 2019-12-31 分类于 The Universe

每一个Logo，都有它的一段故事

每一段故事，都代表着每次我做出的选择

每个选择，也都付出了相应的后果……

1.1

  • 闲来无事，做个什么好呢？
  • 试试创造一下自己的小天地吧。
  • “毕竟我也下决心改变自己的脾气了呢”。

    2.1

  • “我应该是时候去尝试寻找一些我所未尝试的事物了吧，毕竟我的世界目前还是一片黑白。”

    2.2

  • “我所定好的目标，渲染出了我已有的色彩，我的道路，明晰可见。”

    4.1

  • “故事刚刚开始。”我仍然面对着一片未知的空白。

    4.2

  • “或许故事远不会结束吧。”我对自己说。

    3.1

  • “我的色彩在哪里呢？找个人去填充一下吧。”

    3.2

  • “我的色彩，是你给的。”

    3.3

  • 我尽量降低自己的个人色彩，加了个白边框，但你觉得只有你给的色彩才是最好的。

    5.0

  • “那么，就让一切回归到原点吧。”
