要求：

  • First: 你会用电脑
  • Second: 你会装Python并且会添加环境变量

接下来是正文

  1. 在以上条件都具备之后，打开cmd，执行pip install flask
  2. 新建一个后缀名为.py的文件
  3. 打开.py这个文件

  4. 写入

    from flask import Flask

    app = Flask(name)

@app.route('/')
def index():
    return 'fxckUJava'


if __name__ == '__main__':
    app.run()
